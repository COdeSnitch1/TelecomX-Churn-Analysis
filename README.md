# 📡 Telecom Churn Analysis

## 🎯 Objective
Develop a predictive model to identify customers at risk of churn in a telecommunications company.

## 📊 Dataset
Simulated telecom dataset including:
- Contract type
- Monthly charges
- Tenure
- Support calls
- Internet service
- Payment method
- Data usage
- Churn (target variable)

## 🔎 Methodology
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature encoding and scaling
4. Logistic Regression model
5. Random Forest model
6. Hyperparameter tuning (GridSearchCV)
7. Model evaluation (ROC-AUC)

## 📈 Results
The Random Forest model achieved strong predictive performance with optimized hyperparameters.

## 💡 Business Insights
- Monthly contracts show higher churn probability.
- High number of support calls increases risk.
- Early retention strategies recommended for new customers.
